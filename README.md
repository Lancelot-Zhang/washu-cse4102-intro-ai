# CSE 4102: Introduction to Artificial Intelligence

Course archive for **CSE 4102: Introduction to Artificial Intelligence** at Washington University in St. Louis.

This repository collects selected course materials and project archives for personal study, review, and portfolio documentation.

## Repository Contents

- `Practice Exam 1.pdf` and `Practice Exam 2.pdf`: practice exam materials
- `Project 0.zip` to `Project 3.zip`: archived course project packages

## Topics

This course repository is related to core artificial intelligence concepts, including search, reasoning, planning, probabilistic models, and applied AI problem solving.

## Notes

The materials in this repository are maintained as a personal academic archive. If you are currently taking the course, please follow your institution's academic integrity policy and use these files only as appropriate reference material.
